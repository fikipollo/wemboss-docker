wEMBOSS: a web interface for EMBOSS

Welcome to wEMBOSS!

wEMBOSS is a simple and powerful interface which can be installed in 5 min. EMBOSS programs are (Rice et al., 2000) typically accessed from the command line or console. wEMBOSS provides a user-friendly way to access all EMBOSS programs, with tools that help to organize work and review it later.

wEMBOSS is an Open Source software released under the GPL license as well as EMBOSS.
